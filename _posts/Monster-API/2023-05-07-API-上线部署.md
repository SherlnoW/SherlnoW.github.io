[//]: # (### 后端部署)

[//]: # ()
[//]: # (1. 先部署服务器的数据库导入sql语句)

[//]: # (2. 开放服务器3306端口，数据库权限运行所有ip访问)

[//]: # (3. 修改dubbo配置host地址为服务器**内网地址**，向nacos注册服务)

[//]: # (```yml)

[//]: # (## 配置)

[//]: # (dubbo:)

[//]: # (  registry:)

[//]: # (    address: nacos://服务区内网地址:8848)

[//]: # (## 外网地址不能会导致服务不能注册)

[//]: # (```)

[//]: # (4. 修改cookie的作用域)

[//]: # (```yml)

[//]: # (server:)

[//]: # (  address: 0.0.0.0)

[//]: # (  port: 9092)

[//]: # (  servlet:)

[//]: # (    context-path: /api)

[//]: # (    # cookie 30 天过期)

[//]: # (    session:)

[//]: # (      cookie:)

[//]: # (        max-age: 2592000)

[//]: # (        domain: 服务器外网IP)

[//]: # (```)

[//]: # (5. 修改SDK中的网关IP为服务器地址)

[//]: # (6. 启动端口)

[//]: # (```)

[//]: # (java -jar MonsterAPI-backend-0.0.1-SNAPSHOT.jar )

[//]: # (```)
